# URL_shortener

## 功能
* 使用者可以輸入網址轉換成短網址
* 短網址可以直接連接到原始網址
* 空白輸入警示&阻擋
* 可以在網址列後方貼上短網址連接到原始網址


## 需求
* node.js: v14.17.6
* express: v4.17.1
* express-handlebars: v5.3.4
* mongoose: v6.0.10
* MongoDB - 第一次使用請先確保資料庫內沒有名為"ShortUrl"的資料庫，如果有存在同名的資料庫會導致出錯

## 安裝方法
* 執行Terminal，透過npm安裝URL_shortener
* 下載專案  
```
  git clone https://github.com/azod2/URL_shortener
```
* 安裝
```
cd URL_shortener
npm install
```
* 執行
```
npm run dev
```
* 在瀏覽器輸入網址
```
http://localhost:3000
```
